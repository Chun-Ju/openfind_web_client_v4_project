# openfind_web_client_v4_project
請撰寫一個程式來複製指定網站內所有的網頁資料

#該版本 僅為支援一個pdf網址的版本測試用
##並僅提供https的連結服務,http暫時不提供
##也並未提供從抓下的網頁中href的分析
